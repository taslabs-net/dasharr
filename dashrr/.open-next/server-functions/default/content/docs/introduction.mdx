# Introduction

Dasharr is a modern dashboard framework that allows developers to build beautiful, responsive dashboards with minimal effort.

The goal of Dasharr is to be simple and developer-friendly. There are other dashboard solutions like Grafana or Metabase, but they often require complex setups and configurations for custom use cases.

Additionally, what makes Dasharr special is that it focuses on real-time data visualization with a component-first approach, which means you can build custom widgets and layouts easily. Some developers might prefer traditional BI tools at first, but we believe component-based dashboards are the future, and once you try building with Dasharr, you'll appreciate the flexibility and control it provides. For example, you can now create custom visualizations that perfectly match your brand and data requirements.

## Key Features

- **Component-Based Architecture**: Build dashboards using reusable components
- **Real-Time Updates**: Live data streaming and automatic refresh
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile
- **Easy Integration**: Connect to any data source with our flexible API
- **Customizable Themes**: Match your brand with custom styling
