## Configuration Guide

### Required Settings

- **Server URL**: The URL of your Plex server (e.g., `http://192.168.1.100:32400` or `https://plex.yourdomain.com`)
- **Token**: Your Plex authentication token for API access

### Finding Your Plex Token

1. Open Plex Web App
2. Navigate to any media item
3. Click "Get Info" â†’ "View XML"
4. Look for `X-Plex-Token` in the URL

### Alternative: Environment Variables

You can also configure Plex using environment variables:

```bash
PLEX_URL=http://192.168.1.100:32400
PLEX_TOKEN=your_plex_token_here
```
