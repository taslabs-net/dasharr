## Configuration Guide

### Required Settings

- **Server URL**: The URL of your Jellyseerr instance (e.g., `http://192.168.1.100:5055` or `https://jellyseerr.yourdomain.com`)
- **API Key**: Your Jellyseerr API key for authentication

### Finding Your API Key

1. Open Jellyseerr web interface
2. Go to Settings â†’ General
3. Find the API Key section
4. Click "Copy" to copy your API key

### Alternative: Environment Variables

You can also configure Jellyseerr using environment variables:

```bash
JELLYSEERR_URL=http://192.168.1.100:5055
JELLYSEERR_API_KEY=your_api_key_here
```
