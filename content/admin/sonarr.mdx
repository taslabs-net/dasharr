## Configuration Guide

### Required Settings

- **Server URL**: The URL of your Sonarr instance (e.g., `http://192.168.1.100:8989` or `https://sonarr.yourdomain.com`)
- **API Key**: Your Sonarr API key for authentication

### Finding Your API Key

1. Open Sonarr web interface
2. Go to Settings â†’ General
3. Copy the API Key

### Alternative: Environment Variables

You can also configure Sonarr using environment variables:

```bash
SONARR_URL=http://192.168.1.100:8989
SONARR_API_KEY=your_api_key_here
```
