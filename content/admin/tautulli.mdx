## Configuration Guide

### Required Settings

- **Server URL**: The URL of your Tautulli instance (e.g., `http://192.168.1.100:8181` or `https://tautulli.yourdomain.com`)
- **API Key**: Your Tautulli API key for authentication

### Finding Your API Key

1. Open Tautulli web interface
2. Go to Settings â†’ Web Interface
3. Copy the API Key

### Alternative: Environment Variables

You can also configure Tautulli using environment variables:

```bash
TAUTULLI_URL=http://192.168.1.100:8181
TAUTULLI_API_KEY=your_api_key_here
```
